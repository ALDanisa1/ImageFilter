1. To get more information about the project go through the report pdf file
2. To run the program use the command "filename.java inputImage.jpg outputImage.jpg windowSize"
3. The filename is java file name you want to run, inputImage is the image you want to filter, outputImage
is the image name you want to save to and windowSize is the size of portion you want to filter with throughout the image
4. example command   "MedianFilterSerial.java sample.jpg output.jpg 5"
